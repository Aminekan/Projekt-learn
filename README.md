# Projekt-Learn – Webbasierte Lernplattform

Projekt-Learn ist eine webbasierte Lernplattform, die Lernorganisation, aktives Üben und Motivation kombiniert.  
Die Website wurde im Team entwickelt und richtet sich an Lernende, die ihren Lernprozess strukturieren und verbessern möchten.

---

## Projektziele

- Lernaufgaben übersichtlich planen (LearnPlan)
- Vokabeln effektiv lernen und wiederholen
- Lernfortschritt sichtbar machen (Statistiken und Diagramme)
- Motivation durch ein integriertes Mini-Game steigern
- Umsetzung mit HTML, CSS und JavaScript (ohne Backend)

---

## Funktionen und Seiten

### Home
- Startseite der Anwendung
- Navigation zu allen Modulen
- Einheitliches Design und Layout

### LearnPlan
- Lernplan- und Aufgabenverwaltung
- Aufgaben anlegen, bearbeiten und als erledigt markieren
- Übersicht über geplante Lernaktivitäten

### Vokabeltrainer
- Vokabeln anlegen (Wort und Übersetzung)
- Lernmodus mit Abfrage
- Bewertung: richtig / falsch
- Speicherung der Daten mit LocalStorage

### Learnfortschritt
- Anzeige von Lernstatistiken
- Anzahl der Vokabeln
- Richtige und falsche Antworten
- Diagramm der letzten Lerntage

### Mini-Game
- Spielerisches Lernen (z. B. Memory- oder Quiz-Spiel)
- Nutzung der gespeicherten Vokabeln
- Motivation durch Zeit, Punkte oder Bestzeiten

---

## Verwendete Technologien

- HTML5 – Struktur der Website  
- CSS3 – Design und Layout  
- JavaScript (Vanilla) – Logik und Interaktion  
- LocalStorage – Speicherung der Daten im Browser

## Installation und Nutzung

1. Repository klonen oder herunterladen  
2. Datei `index.html` im Browser öffnen  
3. Vokabeln im Vokabeltrainer anlegen  
4. Lernplan, Lernmodus, Fortschritt und Mini-Game nutzen  

Hinweis: Es ist kein Server oder Backend notwendig.
